# NFS Car Project

Car performance rankings for Need for Speed games.

Personal project comprehensively analyzing every car in each NFS game with the available game stats/specs and specific game mechanics.
Mainly a passion project.


## Games covered

| Game | Cars | Charts |
|------|------|--------|
| Need for Speed: Most Wanted (2005) | 32 | 14 |

More games will be added over time.
